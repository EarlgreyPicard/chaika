chaika リリースノート

chaika が新しいバージョンへと更新されました。

ご利用ありがとうございます。
公式サイト等へのリンクはページ下部をご覧ください。

1.6.0 - 2013/02/11

  • p2経由での書き込みに対応

    初めてp2経由でBeを有効にして書き込む場合、p2.2ch.net にて設定する必要があります。

  • リリースノートページの追加
  • 更新後初回起動時にリリースノートを自動で表示するようにした
  • 書き込みプレビューを表示するかどうかの設定を追加
  • 以下の場合に警告するかどうかの設定を追加
    • ログを削除する時
    • 書き込みウィザードを閉じる時
    • fusianasan
    • Beが有効な時 (Be関連板を除く)
    • p2が有効な時
  • Be関連板で書き込む際に自動的にBeを有効にする機能を追加
  • 書き込みが終わるごとにBe, p2を無効にする機能を追加
  • ログマネージャ、スレッド一覧、トラブルシューティング情報にfaviconを追加
  • Be、 p2、●のパスワードをログインマネージャで保管するように変更

    これまでパスワードは平文で保存されていましたが、ログインマネージャを使うことにより暗号化された状態で保存されるようになり、 セキュリティが大幅に向上しました。

    なお、従来の保存先である extensions.chaika.maru_password / extensions.chaika.login.be.password / extensions.chaika.login.p2.password は空文字のまま残りますが、これは内部処理で使用しているためであり不具合ではありません。

  • スレッド一覧の「購読中」ボタンを「既読」ボタンへと名称を変更
  • Beのアカウントの設定を設定画面から行うように変更
  • defaultmail.txtで空白以外が指定されている板のスレッドへ書き込む際は、ウィザードのsageチェックボックスの値を記憶しないよう変更
  • 設定ウィンドウのサイズを内容に応じて変化するように変更
  • 範囲が非常に大きいアンカーを含むレスがあるとメモリ不足になってスレッドが読み込めなくなる問題を修正
  • Firefox 18以降でトラブルシューティング情報が動かない問題を修正
  • 書き込みウィザードでタイトルの実体参照が解除されていない問題を修正
  • 書き込みウィザードのプレビュー画面で「"」が不要に実体参照へ変換されていた問題を修正
  • defaultmail.txtで設定されたメールアドレスと書き込みウィザードのsageチェックボックスの値が矛盾することがある問題を修正

1.5.8 - 2013/02/02

  • 「最近アクセスしたスレッド」等が新しいタブで開くかどうかについて設定を反映するようにした
  • SETTING.TXTに板名が書かれていない板ではページタイトルを板名に設定するようにした
  • Content-Typeが不明なスキンのファイルはtext/plainとして送信するようにした
    Web Fontなど、従来Internal Server Errorとなっていたファイルがスキンで使えるようになりました。
  • dat落ちしたスレッドを表示した時にログが二重になることがある問題を修正
  • したらばやまちBBSの透明削除があるスレッドをログピックアップモードで表示すると表示されるレスがずれる問題を修正
    本修正に伴い、透明削除があるスレッドのdatが過去のchaikaと非互換になりました。表示がおかしい場合にはdatを再取得して下さい。
  • ログピックアップモードの時に新着を取得しない問題を修正
  • 連鎖あぼーんが正しく動かないことがある問題を修正
  • アンカー修正機能が有効な時にアンカーが誤って表示されることがある問題を修正
  • その他軽微なバグの修正

1.5.7 - 2013/01/14

  • したらばでの新規スレッド作成機能を実装
  • 過去ログ取得時のアラートにchaikaのアイコンを表示するようにした
  • HTML版の2ch検索を実装
    about:configよりextensions.chaika.bbsmenu.find2ch.use_rssで従来のRSSを用いた検索と切り替えることができます(デフォルトはRSS検索)。
  • 不正なアンカー（全角＞、カンマ区切りなど）を自動で修正する機能を実装
    本機能は実験的機能となっています。
    about:configよりextensions.chaika.thread_fix_invalid_anchorで切り替えることができます(デフォルトOFF)。
  • maxVersionをFirefox 18に変更
  • minVersionをFirefox 10に変更
  • Be IDがあるときにIDの末尾に余計なスペースが付加される問題を修正
  • したらばやまちBBSの透明削除があるスレッドを開くとレス番がずれる問題を修正
  • ID: の前に文字があるとIDのリンクが貼られない問題を修正
  • 特定環境下でスレッドの読み込みに時間がかかることがある問題を修正
  • ツールバーアイコンが消えることがある問題を修正
  • 大きいアイコンにするとWindowsでツールバーボタンに画像が表示されなくなる問題を修正
  • 範囲指定のあるアンカー等で正しく連鎖あぼーんが働かない問題を修正
  • Firefox 18以降でプロキシを「ブラウザの設定に従う」以外にするとスレッドが開けなくなる問題を修正
  • Firefox 18以降で「直接接続する」が動作しない問題を修正
  • Firefox 20以降でスレッド一覧が開けない問題を修正
